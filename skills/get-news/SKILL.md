# 晨间新闻聚合

## 用途

每日自动聚合高价值新闻信息，帮助用户在短时间内（如地铁通勤）掌握：
- 前一日至当天的世界局势动态
- 中东地区具体事态发展
- 中美欧等主要经济体货币政策、经济政策
- AI/AI Agent/Agent Memory 技术前沿
- 学术成果与工程应用

### 真实性校验机制

1. **多源交叉验证**：重要新闻至少 2 个独立信源确认
2. **来源可信度评分**：优先官方机构、权威媒体
3. **时间戳校验**：标注信息获取时间，排除过时信息
4. **标签标注**：
   - [已验证] - 多源确认
   - [单一信源] - 仅一个来源，需进一步关注
   - [官方发布] - 政府/机构官方发布
   - [技术分析] - 专家/机构分析观点

## 工作流程

### 自动模式（推荐）

每天 7:30 自动执行，生成当日晨报：

```
1. 获取前一日 18:00 至今日 7:30 的新闻
2. 按类别聚合去重
3. 交叉验证重要新闻
4. 生成结构化 Markdown 报告
5. 输出到终端/保存到文件
```
### 时区
所有内容的时间均是Aisa/Shanghai 时间。


### 手动模式

随时获取指定时间范围的新闻：

```
# 获取今天的新闻
/morning-news

# 获取指定日期范围
/morning-news --start 2026-04-01 --end 2026-04-03

# 只获取特定类别
/morning-news --category ai,economy
```

## 使用方式

### 方式一：定时自动（推荐）

已配置每天 7:30 自动执行，直接查看输出即可。

### 方式二：手动触发

```bash
# 执行完整晨报
python scripts/morning_news.py

# 只获取今天的新闻（缩短范围）
python scripts/morning_news.py --today

# 获取指定日期
python scripts/morning_news.py --date 2026-04-02

# 指定类别
python scripts/morning_news.py --category world,economy,ai
```

### 方式三：通过 Claude 触发

```
获取今日晨报
同步晨间新闻
/morning-news
```

## 配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 执行时间 | 7:30 | 可修改定时任务 |
| 时间范围 | 前日18:00-当日7:30 | 覆盖昨日晚间+今晨 |
| 输出方式 | 终端 + 文件 | 文件保存在 `reports/` |
| 信息条数 | 每类 5-10 条 | 避免信息过载 |

## 依赖安装

```bash
cd morning-news
pip install -r requirements.txt
```

依赖：
- `feedparser` - RSS 解析
- `requests` - API 调用
- `beautifulsoup4` - 网页抓取
- `python-dateutil` - 时间处理
- `rich` - 终端美化输出

## 注意事项

1. **网络要求**：需要访问国际网络获取 Reuters、Hacker News 等
2. **API 限制**：部分源有频率限制，请勿过于频繁调用
3. **信息时效**：新闻具有时效性，建议每日查看
4. **真实性**：虽然有多重校验，但重要决策仍需核实原始来源
